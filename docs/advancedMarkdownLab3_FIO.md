## Сноска

Markdown полезен в разработке[^1].

[^1]: Markdown поддерживается на GitHub, GitLab, в большинстве редакторов кода и систем документирования.

## Alert-блоки

> [!TIP]
> Для экранирования специальных символов в Markdown используйте обратный слэш `\`. Например, чтобы написать *курсив* без форматирования, напишите `\*курсив\*`.

> [!IMPORTANT]
> Всегда проверяйте статус файлов перед коммитом с помощью `git status`.

## Escape-символы

- Заголовок без форматирования: `\# Это не заголовок`
- Звёздочка как текст: `\*просто звёздочка\*`
- Подчёркивание без курсива: \_не курсив\_

## HTML в Markdown

```
<p style="background-color: #8564ab; padding: 15px; border-radius: 8px; border-left: 4px solid #4f009e;">
<strong>Обратите внимание:</strong> Этот параграф оформлен с помощью <u>HTML-тегов</u> внутри Markdown-документа. Markdown полностью поддерживает встраивание HTML, что позволяет использовать сложное форматирование там, где возможностей Markdown недостаточно.
</p>
```
