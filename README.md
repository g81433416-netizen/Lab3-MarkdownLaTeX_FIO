# Лабораторная работа №3: Markdown и LaTeX

## 📋 Краткое описание
Данный проект демонстрирует практическое применение языка разметки Markdown и системы вёрстки LaTeX для создания структурированной документации. В ходе работы были изучены основные элементы форматирования, работа с Git и публикация проекта на GitHub.

### Форматирование текста
- **Полужирный текст**
- *Курсив*
- ***Полужирный курсив***
- ~~Зачёркнутый текст~~
- `Моноширный текст`

### Списки

#### Маркированный список
- Пункт 1
- Пункт 2
- Пункт 3

#### Нумерованный список
1. Первый пункт
2. Второй пункт
3. Третий пункт

#### Вложенный список
1. Основной пункт
   - Подпункт 1
   - Подпункт 2
     1. Подподпункт 1
     2. Подподпункт 2

### Цитата
> Markdown — это простой язык разметки, который позволяет быстро оформлять документацию, не отвлекаясь на сложный HTML-код.
